### 🧠 문제 설명

하샤드 수 인지 판별. 양의 정수 x가 하샤드 수이려면 x의 자릿수의 합으로 x가 나누어져야 합니다. 예를 들어 18의 자릿수 합은 1+8=9이고, 18은 9로 나누어 떨어지므로 18은 하샤드 수입니다.

### 💡 아이디어

---list(str()): 숫자를 문자로 변환해서 list에 저장
